Веб-застосунок онлайн-аптеки, розроблений на фреймворку Django в рамках навчальної дисципліни «Розроблення програмного забезпечення» (КНУ ім. Тараса Шевченка, кафедра Теоретичної кібернетики).

Функціонал

Каталог ліків — перегляд препаратів з фільтрацією за категорією, типом та текстовим пошуком
Пошук за симптомами — підбір ліків за симптомами через теги або текстовий запит
Картка препарату — детальна інформація: склад, дозування, протипоказання, симптоми
Кошик — додавання товарів, зміна кількості, видалення (зберігається в сесії без реєстрації)
Рецептурні препарати — автоматичне розпізнавання та обов'язкове підтвердження рецепту при оформленні
Оформлення замовлення — вибір способу доставки: Нова Пошта, Укрпошта, кур'єр, самовивіз
Адмін-панель — управління каталогом, замовленнями та статусами


Технологічний стек
Backend - Python 3.10+, Django 4.2 
База даних - SQLite 3 
Frontend - Bootstrap 5.3, Bootstrap Icons
Шаблонізатор - Django Template Engine
Кошик - Django Sessions
Адмінка - Django Admin

Встановлення та запуск
1. Клонуй репозиторій
bashgit clone https://github.com/your-username/pharmacy-project.git
cd pharmacy-project
2. Встанови залежності
bashpip install -r requirements.txt
Якщо pip не знайдено — спробуй pip3 або python -m pip
3. Застосуй міграції
bashpython manage.py migrate
4. Заповни базу тестовими даними
bashpython manage.py seed_data
5. Запусти сервер
bashpython manage.py runserver
6. Відкрий у браузері

Сайт → http://127.0.0.1:8000/
Адмінка → http://127.0.0.1:8000/admin/
